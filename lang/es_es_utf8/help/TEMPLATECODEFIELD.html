Identificador de página escaneada

En este campo debe introducir el código que identifica la página que está viendo.

Este código debe aparecer impreso al lado del código de barras de la esquina superior derecha del cuestionario, siempre que se haya seleccionado la opción "Mostrar código legible para humanos" a la hora de generar el documento PDF original.

El valor que debe introducir en el campo son todos los dígitos que observe incluyendo el último que indica el número de página dentro del cuestionario.

Ejemplo:
Número impreso en el código de barras: 3231
Rellene en el campo = 3231
