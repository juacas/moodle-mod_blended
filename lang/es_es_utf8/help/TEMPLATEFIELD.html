Activitycode

En este campo debe introducir el código que identifica al cuestionario.

Este código debe aparecer impreso al lado del código de barras de la esquina superior derecha del cuestionario, siempre que se haya seleccionado la opción "Mostrar código legible para humanos" a la hora de generar el documento PDF original.

El valor que debe introducir en el campo son todos los dígitos que observe, menos el último, que indica el número de página del cuestionario.

Ejemplo:
ID: 3231
ACTIVITYCODE = 323
