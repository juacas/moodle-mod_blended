Generar cuestionarios en papel

En la primera parte de la página podemos ver una tabla con los trabajos realizados o que se encuentran en espera. El profesor podrá eliminar de la base de datos los trabajos pinchando sobre el icono con una X que se encuentra al lado de cada tarea.
Cada trabajo se encuentra identificado por el usuario que lo generó y su fecha de creación.

En la segunda parte de la página se muestra un formulario que le permitirá seleccionar una serie de parámetros de configuración para el cuestionario:

Formato de PaperQuiz
Seleccione el cuestionario: Elija de la lista el cuestionario que se desea generar en PDF.
Para que esta lista esté disponible, al menos debe existir un cuestionario con preguntas en el curso al que pertenece el módulo blended.
Para crear un cuestionario con preguntas, acuda al menú principal del curso, y seleccione Agregar Actividad → Cuestionario.

El módulo Blended soporta hasta cuatro tipos diferentes de preguntas en un mismo cuestionario:
- Verdadero/Falso.
- Descripción.
- Preguntas multichoice, con una o varias respuestas posibles.
- Preguntas de tipo ensayo.

Número de ejemplares: Indica el número de cuestionarios que se imprimirán en un mismo documento PDF.
Los cuestionarios se imprimirán de manera consecutiva.

Generar cuestionario más tarde: Al marcar esta casilla, el trabajo se ejecutará más tarde.
El proceso cron() de Moodle creará los documentos en unos minutos. Se recomienda usar esta opción cuando se desean generar un número alto de cuestionarios.

Formato de las etiquetas.
Identificar las etiquetas para los humanos.
No identificar. En los códigos 2D el id del cuestionario irá codificado en el código y no podrá ser legible para los humanos.
Mostrar código legible para los humanos. En las etiquetas aparecerá un número de identificación que podrá ser leído visualmente.

Para efectuar la generación del documento PDF pulse el botón "Generar ficheros para el cuestionario".
