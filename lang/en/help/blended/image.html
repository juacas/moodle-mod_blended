Imágenes de detección

La imagen observada corresponde con una página procesada del cuestionario.

En la esquina superior derecha del cuestionario debe aparecer un código de barras. Este código identifica al cuestionario. Siempre que se haya seleccionado la opción "Mostrar código legible para humanos" a la hora de generar el documento PDF original, debe aparecer un número impreso al lado del código de barras de este código del cuestionario. El número que puede leer en dicha etiqueta indica el código del cuestionario (ActivityCode) y el número de página dentro del cuestionario.

Ejemplo:
ID: 3231 significa
ACTIVITYCODE = 323
Número de página = 1

En el encabezado del cuestionario debería de aparecer un código de barras. Este código es la etiqueta que el alumno tiene que adherir al cuestionario, en cada página del mismo.
Debe de aparecer un código identificando al alumno.

Las marcas detectadas aparecen indicadas con un círculo y una flecha en color rojo. La disposición de respuestas sigue el patrón que aparece a continuación:

A B

C D
