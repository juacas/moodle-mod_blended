Generar página de etiquetas

Esta sección permite generar un documento PDF que contenga etiquetas en formato código de barras para que los alumnos identifiquen unívocamente sus tareas, cuestionarios o exámenes.

Cómo generar un documento con etiquetas

En primer lugar, configure el formato de la página que desea imprimir:

Formato de Página:
- Seleccione el formato de papel que va a utilizar. (Por defecto se empleará el formato estándar Din-A4)
- Puede ajustar los márgenes de la página de forma individual.
- El número de filas y de columnas corresponden a las dimensiones de la tabla que se generará en la página. En cada celda de la tabla aparecerá una etiqueta.
Las dimensiones de las etiquetas, por tanto, se ajustarán al tamaño de celda según el número de filas y de columnas indicadas.


A continuación seleccione los alumnos para los que desea imprimir las etiquetas y marque si desea que aparezca o no visualmente el código de cada etiqueta de forma legible:

Métodos para la disposición de las etiquetas
Una etiqueta por cada estudiante activo en el curso: Se imprimirá una etiqueta por cada estudiante que al menos haya entrado una vez al curso
Una etiqueta por cada estudiante matriculado en el curso: Se imprimirá una etiqueta por cada estudiante matriculado, independientemente de que éste se haya conectado o no al curso.
Páginas completas de etiquetas para los estudiantes seleccionados en la lista de la izquierda: Seleccione los alumnos para los que desea imprimir las etiquetas en la lista de la izquierda. Se imprimirá una página de etiquetas por cada alumno seleccionado.


Formato de las etiquetas
No identificar: En las etiquetas no aparecerá ningún tipo de texto legible, únicamente el código de barras.
Mostrar código ID legible: En cada etiqueta aparecerá un número identificativo de cada alumno. Este número depende de la opción que esté seleccionada en el parámetro Identificador de las opciones de configuración del módulo. (Vea Actualizar [Nombre del módulo] si dispone de los permisos suficientes).
Dicho parámetro puede tomar los siguientes valores:
- Id de usuario: Identificador interno de Moodle para cada usuario.
- Número de identificación: Campo ID del perfil de usuario, normalmente debería ser su DNI.
- Identificador personalizado: Campo de identificación personalizado del perfil de usuario. Únicamente aparecerá esta opción si un administrador ha creado esta característica para el módulo.
Mostrar nombre completo del usuario: Mostrará nombre y apellidos de cada persona en la etiqueta.
