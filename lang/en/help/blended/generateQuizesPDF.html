Documentos PDF generados

Si el proceso de generación de cuestionarios ha tenido éxito podrá realizar dos acciones:
- Abrir la carpeta que contiene el documento PDF, siguiendo el enlace que aparece en la página (debe tener permiso para realizar esta acción).
- Descargar directamente el documento, utilizando el enlace de descarga directa.
En caso de que haya habido algún error se le mostrará un mensaje notificándolo.
