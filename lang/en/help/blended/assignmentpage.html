Generar página de tarea

Esta sección le permitirá imprimir una página en PDF a partir de una tarea Moodle. Es una alternativa a pegar la etiqueta adhesiva con el código de barras en un folio en blanco y escribir a mano el resto de campos que vienen impresos en la hoja de tarea. Dispone de una lista desplegable para seleccionar la tarea de entre todas las disponibles en el curso. También hay un campo de texto opcional para introducir un título libre alternativo para la hoja de tarea.

Para poder imprimir una hoja de tarea, previamente se debe haber creado dicha tarea en el curso y configurado sus características, mediante la opción de edición Agregar Actividad → Tareas.

Si se accede como profesor además se muestra una lista desplegable con el listado de todos los alumnos matriculados en el curso, para que pueda elegirse el alumno del cual se imprimirá su hoja de tarea.
